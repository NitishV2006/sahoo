# Nitish Raj | Elite Full-Stack Architect

A high-octane, cinematic portfolio experience inspired by the aesthetic of high-budget action cinema (Saaho) and digital "System Breach" narratives. This project is not just a resume; it's a digital performance.

## 🎬 The Experience

The portfolio is structured as a multi-scene narrative that transitions from a "Fake Identity" decoy to a "Real Identity" reveal through a simulated system breach.

### Key Scenes
1. **The Hook (`SILENCE` / `SEARCHING`)**: A minimalist, high-tension entry point that challenges the user to find the "real" developer.
2. **The Terminal (`TERMINAL`)**: A hacker-style command-line interface that simulates a security bypass.
3. **The Decoy (`FAKE_IDENTITY`)**: A standard, "boring" profile used as a narrative foil for the real reveal.
4. **The Breach (`BREAK_EVENT`)**: A high-intensity glitch sequence featuring:
    - **Matrix Red Rain**: Cascading digital characters.
    - **RGB Split**: Visual channel separation and screen skewing.
    - **Static Noise**: Simulated monitor failure.
5. **The Reveal (`NAME_REVEAL` / `IMAGE_REVEAL`)**:
    - **Cinematic Flash**: High-brightness screen flashes on key reveals.
    - **Light Sweep**: A metallic glint that slides across the name.
    - **Audio Impact**: Custom-synthesized dramatic thuds using the Web Audio API.
6. **Core Architecture (`SKILLS`)**: An interactive, bento-style grid of technical proficiencies with hover-triggered icons and lift effects.
7. **Final Climax (`FINAL_CLIMAX`)**: A high-stakes call to action with a pulsing, glowing "Initiate Contact" button.

## 🛠️ Tech Stack

- **Frontend**: React 18+ with TypeScript.
- **Styling**: Tailwind CSS (Utility-first, high-performance styling).
- **Animations**: `motion` (Framer Motion) for layout transitions and complex sequencing.
- **Icons**: `lucide-react` for a consistent, technical iconography.
- **Audio**: Web Audio API for real-time sound synthesis and HTML5 Audio for background scores.

## ⚡ Special Features

### 🔊 Audio Synthesis
The "Reveal Impact" sound is generated programmatically using the browser's `AudioContext`. It combines a low-frequency sine wave ramp (the "thud") with a high-frequency triangle wave (the "shimmer"), ensuring zero-latency dramatic feedback.

### 🎨 Custom CSS Animations
- `animate-light-sweep`: A sophisticated linear-gradient mask that creates a "Saaho-style" metallic glint.
- `animate-pulse-glow`: A multi-layered box-shadow animation for the final CTA.
- `animate-glitch-skew`: A rapid transform-based skew that simulates digital instability.

### 🧩 Responsive Design
The experience is fully responsive, scaling from ultra-wide desktop monitors to mobile devices, ensuring the cinematic impact is preserved on all screens.
